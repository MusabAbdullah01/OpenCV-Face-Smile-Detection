# Real-Time Face & Smile Detection using OpenCV

This project implements an interactive computer vision application that detects human faces and captures smiling moments in real-time. It leverages pre-trained Haar Cascade classifiers to identify facial features directly from a live video stream.

## 🚀 Key Features
* **Real-Time Video Processing:** Capture and process live video streams with minimal latency.
* **Dual-Stage Classification:** Detects human faces first, then restricts the search area to track smiles dynamically.
* **Visual Annotations:** Draws reactive bounding boxes around facial landmarks (Blue for faces, Green for smiles).
